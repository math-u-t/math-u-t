## プログラミングを始めたきっかけは？

最初に触れたのは「**Scratch**」でした。  
Scratchはビジュアルプログラミング言語で、画像編集エディターや簡単なシステムがすぐに作れる仕組みが整っており、初心者でも楽しみながらプログラムの基礎を学ぶことができます。  
そのため、今でもScratchをよく使っており、プログラムを書く楽しさを実感できる場として、非常に大切な存在です。
